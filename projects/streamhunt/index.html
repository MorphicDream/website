StreamHunt

Minecraft plugin to allow stream viewers to interact with the stream with killer zombies

Overview

This is a minecraft plugin that allows chatters on a stream to use /hunt {player} and it will generate a zombie with the chatters name, it will hunt whoever the chatter typed.

Platform: Minecraft

Details

Once fully tested ill put the plugin onto spigot and probably add metrics. I'm going to also code small bots that will be the bridge between the stream platform and the mine-craft server. I think the best choices of stream platforms at this stage are probably, Twitch, YouTube and maybe Kick. There will be a config allow you to decide if you are going to stream on one platform or more than one.
