Infected Config

Details of the config for Infected plugin

Overview

This shows the format used in the config for the Infected plugin

Platform: YML (config)

Config Layout


      #Sets if the infection should be started with commands - true means using the /infection stop/start command
      #If set to false the infection will start when one player is online and stop when there is no-one online
      ManualControl: false

      #Sets the maximum limit for infected zombies across the world
      MaxInfectedZombiesSpawned: 20

      #A global time in seconds for each phase to last
      TimeBetweenPhases: 60

      #Setting this to true means each phase will be the same as TimeBetweenPhase
      #Setting this to false means you need to set the three phase times
      UseSingularPhaseTime: true

      #These will set the times for each phase individually in seconds(60 = 1 minute)
      PhaseOneTime: 60
      PhaseTwoTime: 60
      PhaseThreeTime: 60

More details

ManualControl is a boolean that will enable or disable the commands used to stop and start the infection. By default, it's set to false as it's mainly used for testing the plugin. But it was suggested that if someone was streaming or had an event they might want to be able to start the infection when everyone was ready.

MaxInfectedZombiesSpawned just sets a limit on how many zombies will spawn, it's set to 20 by default. There is no upper limit really but when in the last phase of the game the zombies spawn in groups of upto four zombies at a time. If there are more players i would be worth upping the limit but for one to two people 20 is more than enough.

TimeBetweenPhases this value needs to be set in seconds and is only used when the UseSingularPhaseTime is set to true, which it will be by default. It just means that every phase will run for the same amount of time.

UseSingularPhaseTime should be set to true if you want the phases to all be the same length of time. When set to false the next values need to be set. This is set to true when created.

PhaseOneTime, PhaseTwoTime and PhaseThreeTime are just to be used when UseSingularPhaseTime is set to false. Each value is based on seconds and default is 60 seconds.
