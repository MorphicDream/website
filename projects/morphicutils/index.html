MorphicUtils

This is a library plugin for java minecraft.

Overview

A very basic library styled plugin to add some simple things easily

Platform: Java Minecraft 1.21.4

Details

The plugin as of now has the ability to do the following
Add a String or Int to an itemstack using persistent data
Get a random String, Int or Double
Convert Strings to Component if using Paper instead of Spigot
Convert Components back to String
Gets the server name from the server.properties file (for use with scoreboards etc)
Send a player to a named server
Boolean check if a specific player is online
Give LivingEntities and Entities the glow effect and being able to check who and what has the effect Allowing the glow effect colour to be changed, only one colour atm
Creating a few particle shapes, circle, sphere, cylinder and a pulsing circle (from inside outwards) and a vertical helix
Menu system including a builder class
Clickable items that can be added to the menu
Basic Hat system using in game blocks (WIP being debugged)
